# Competitive Feature Matrix

Build interactive competitive analysis matrices. Compare products, rate features, and identify market gaps instantly.

## What It Does

- **Add competitors and features** dynamically
- **Rate each product** on every feature: ✅ Has it, 🟡 Partial, ❌ Missing
- **Auto-calculated scores** with visual progress bars
- **Gap Analysis** — automatically identifies your competitive advantages and risks
- **Export to CSV** for sharing with stakeholders
- **Load Demo** to see it in action instantly
- **Persists data** in localStorage — your matrix survives page refreshes

## Built With

- HTML, CSS, JavaScript (no framework)
- localStorage for persistence

## Why I Built This

Every PM does competitive analysis, but it usually lives in a messy spreadsheet. I wanted a tool that makes it visual, interactive, and automatically surfaces insights — specifically identifying where you have an edge and where you're falling behind.
